Class LearnMap

public class LearnMap extends Object implements Mapper<LongWritable,Text,Text,FeatureVector>

LearnMap represents a map node and is composed of two parts: Configuration Configuration loads the movie / user feature vector file into memory to be used on learning Mapping Mapping works over one user / movie / rating tuple, it utilizes their feature vectors and outputs Trained feature vectors.
